# ⚡ 전기회로 & 전자회로 강의 정리

## 1. Mesh Current Method (망 전류법)

### 개념
- **Mesh Current Method**는 이름 그대로 **망(Mesh)** 과 **전류(Current)** 를 이용하는 해석 기법이다.
- 각 메쉬에 **임의의 전류를 직접 설정**하며, 이 임의의 전류가 해석의 기준이 된다.

### 핵심 원리
- 내가 설정한 전류 방향과 **다른 방향의 전류가 존재할 경우**, 내가 설정한 전류가 더 크다고 가정하고 **빼는 형태**로 KVL 방정식을 세운다.
- 메쉬 수 = 미지수 수 = 방정식 수가 되도록 맞춰서 연립방정식을 풀면 된다.

### 📋 풀이 순서

```
1. 각 메쉬에 임의의 전류(I₁, I₂, ...) 설정 (보통 시계 방향)
2. 각 메쉬에 KVL 적용 → 방정식 수립
3. 공유 가지(branch)에서는 (내 전류 - 이웃 전류) 형태로 작성
4. 연립방정식을 크래머 공식 또는 행렬로 풀기
```

---

### 풀이 방법 ① 크래머 공식 (Cramer's Rule)

연립방정식의 미지수가 많아지면 대입법은 너무 복잡해지기 때문에 **크래머 공식**을 활용한다.

#### 예시: 2개의 메쉬

$$
\begin{bmatrix} R_{11} & R_{12} \\ R_{21} & R_{22} \end{bmatrix} \begin{bmatrix} I_1 \\ I_2 \end{bmatrix} = \begin{bmatrix} V_1 \\ V_2 \end{bmatrix}
$$

$$
I_1 = \frac{\det\begin{bmatrix} V_1 & R_{12} \\ V_2 & R_{22} \end{bmatrix}}{\det\begin{bmatrix} R_{11} & R_{12} \\ R_{21} & R_{22} \end{bmatrix}}, \quad I_2 = \frac{\det\begin{bmatrix} R_{11} & V_1 \\ R_{21} & V_2 \end{bmatrix}}{\det\begin{bmatrix} R_{11} & R_{12} \\ R_{21} & R_{22} \end{bmatrix}}
$$

> 💡 **꿀팁:** 크래머 공식에서 분모는 계수 행렬의 행렬식(determinant)이고, 분자는 해당 열을 상수 벡터로 교체한 행렬의 행렬식이다.

---

### 풀이 방법 ② 행렬 (Matrix Method)

$$
[R][I] = [V] \quad \Rightarrow \quad [I] = [R]^{-1}[V]
$$

| 기호 | 의미 |
|------|------|
| $[R]$ | 저항 계수 행렬 |
| $[I]$ | 메쉬 전류 벡터 (미지수) |
| $[V]$ | 전압원 벡터 |

#### 행렬 풀이 단계

```
1. 각 메쉬의 KVL 방정식을 정리
2. [R][I] = [V] 형태로 행렬 구성
3. 역행렬 [R]⁻¹을 구하거나 가우스 소거법 적용
4. [I] = [R]⁻¹[V] 계산
```

> 💡 **꿀팁:** 대각 성분 $R_{ii}$는 해당 메쉬 내 모든 저항의 합, 비대각 성분 $R_{ij}$는 메쉬 $i$와 $j$가 공유하는 저항의 **음수(−)** 값이다.

## 2. 전원의 종류

### 독립 전원 (Independent Source)
- **스스로** 동일한 에너지를 끊임없이 공급하는 전원
- 외부 조건에 무관하게 일정한 전압/전류를 유지
- 예: 이상적인 배터리, 이상적인 전압원

### 종속 전원 (Dependent Source)
- **다른 요소(전압 또는 전류)에 의해** 공급 에너지가 변하는 전원
- 회로 내 다른 변수에 종속됨

### 부하 저항 $R_L$ (Load Resistance)
- **RL = Load Resistance**, 즉 **부하 저항**
- 에너지를 **소비**하는 소자

> 💡 **회로 해석 꿀팁:** 복잡한 회로는 내가 알아보기 쉬운 **등가 회로(Equivalent Circuit)** 로 다시 그리는 것이 핵심이다!

---

## 3. 테브난 등가회로 (Thévenin Equivalent Circuit)

### 개념
- 복잡한 회로 시스템을 **하나의 전압원 $V_{th}$** 와 **하나의 직렬 저항 $R_{th}$** 으로 단순화하는 기법
- 전체 회로가 부하에 공급하는 전압과 전류를 쉽게 파악 가능

```
테브난 등가회로 구조:

    R_th
+--[   ]--+-----> A
|          |
V_th      (부하)
|          |
+----------+-----> B
```

### 📋 테브난 등가 구하는 순서

```
1. 부하(RL)를 분리 (단자 A-B 개방)
2. 내부 전원 제거
   ⚠️  전압원 → Short (단락)
   ⚠️  전류원 → Open (개방)
3. 단자 A-B에서 바라본 등가 저항 R_th 계산
4. 전원을 복원 후, 단자 A-B의 개방 전압 V_th 계산
5. 옴의 법칙으로 부하 전류 계산: I = V_th / (R_th + R_L)
```

## 4. 노턴 등가회로 (Norton Equivalent Circuit)

### 개념
- 테브난과 반대 개념으로, **하나의 전류원 $I_N$** 과 **하나의 병렬 저항 $R_N$** 으로 단순화

```
노턴 등가회로 구조:

+----+----[R_N]----+-----> A
|    |              |
I_N  |             (부하)
|    |              |
+----+-------------+-----> B
```

### 테브난 ↔ 노턴 변환 관계

$$
V_{th} = I_N \cdot R_{th}, \quad R_{th} = R_N
$$

## 5. 중첩의 원리 (Superposition Principle)

### 개념
- 여러 개의 독립 전원이 있을 때, 전원 간의 **상호작용** 때문에 한꺼번에 해석하기 어렵다.
- 전기회로에서의 **수학적 증명은 이미 완료** 되어 있으므로, 우리는 **사용만** 하면 된다.

### 📋 적용 순서

```
1. 전원이 n개라면, 하나의 전원만 남기고 나머지 제거
   - 독립 전압원 → Short
   - 독립 전류원 → Open
2. 남은 전원에 대해 회로 해석 → 전류/전압 계산
3. 모든 전원에 대해 위 과정 반복
4. 각각의 결과를 모두 더함 → 최종 해
```

> ⚠️ **주의:** 종속 전원은 절대 제거하지 않는다! **독립 전원만** 제거 대상이다.

## 6. 반도체 (Semiconductor) 기초

### 반도체 공정 개요

| 구분 | 내용 |
|------|------|
| **설계** | 수십억 개의 트랜지스터를 배치하는 작업 |
| **공정** | 설계된 회로를 실제로 생산하는 작업 |

- 트랜지스터 크기: **나노미터(nm)** 단위
- 이물질(먼지 등) 하나가 심각한 불량을 유발 → 반도체 공장은 **클린룸(Clean Room)** 환경 필수

---

### 🔬 Silicon Wafer (실리콘 웨이퍼)

- 반도체의 핵심 기판 소재
- **순수한 모래 → 규소(Si) 추출 → 웨이퍼 가공**
- 기판은 단단해야 하며, 현재까지 실리콘을 대체할 적합한 물질이 없음

---

### 보어의 원자 모형 (Bohr Atomic Model)

- 원자 구성: **양성자(proton), 중성자(neutron), 전자(electron)**
- 전자는 핵 주위의 **궤도(Shell)** 를 돌며, 각 궤도에 들어갈 수 있는 전자 수는 정해져 있음

| 궤도 번호 | 최대 전자 수 |
|-----------|-------------|
| 1번 (K) | 2개 |
| 2번 (L) | 8개 |
| 3번 (M) | 18개 |
| 4번 (N) | 32개 |

- ⭐ **궤도가 꽉 찼을 때 = 안정 상태** → 전자가 잘 움직이지 않음 → 전기가 잘 통하지 않음
- 주기율표의 **원소 번호 = 전자의 총 개수**

> **예시:** 수소(H, 1번)는 1번 궤도에 전자가 1개뿐 → 안정되려면 1개가 더 필요 → 반응성이 매우 높음

---

### 반도체 물질의 특성

- 반도체에는 **최외각 전자(가전자)가 3~5개**인 물질을 사용
- 실제로 많이 사용되는 물질: **붕소(B), 실리콘(Si), 인(P)**

| 물질 | 특성 |
|------|------|
| 실리콘(Si) | 최외각 전자 4개, 공유 결합 완성 시 부도체와 유사 |
| 붕소(B) | 최외각 전자 3개, p형 도핑에 활용 |
| 인(P) | 최외각 전자 5개, n형 도핑에 활용 |

---

### 에너지 밴드 이론 (Energy Band Theory)

```
에너지 준위 개념도:

전도대 (Conduction Band)  ← 자유 전자가 있는 영역
─────────────────────────
      Band Gap Energy
─────────────────────────
가전자대 (Valence Band)   ← 안정적인 결합 전자 영역
```

| 구분 | Band Gap | 전기 전도성 |
|------|----------|------------|
| 도체 | 매우 작음 (없음) | 매우 잘 통함 |
| 반도체 | 중간 | 조건부로 통함 |
| 부도체 | 매우 큼 | 거의 안 통함 |

- **Band Gap Energy**: 가전자대의 전자가 전도대로 이동하는 데 필요한 에너지
- 반도체는 평소에는 **부도체**처럼 작동하지만, **열 에너지**가 가해지면 결합이 끊어져 **자유 전자**가 생성됨
- 🌡️ 이것이 전자 장비의 **쿨링(Cooling)이 중요한 이유**

---

### Hole (정공)

- 공유 결합에서 전자 하나가 빠져나가면 그 자리에 **빈 자리(Hole)** 가 생김
- 이웃한 전자가 그 자리로 이동 → 마치 Hole이 반대 방향으로 움직이는 것처럼 보임
- Hole은 **양(+)의 전하 운반자**로 취급

---

### 도핑 (Doping)

- 순수 실리콘만으로는 원하는 반도체 특성을 얻기 어려움
- 의도적으로 **불순물(다른 원소)을 첨가**하여 전기적 특성을 조절하는 과정

| 도핑 종류 | 첨가 물질 | 생성되는 운반자 |
|-----------|----------|----------------|
| n형 (n-type) | 인(P), 비소(As) — 전자 5개 | 자유 전자(electron) |
| p형 (p-type) | 붕소(B) — 전자 3개 | 정공(hole) |
