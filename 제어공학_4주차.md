# 제어공학 강의 노트

---

## 1. 물리 시스템 변수의 종류

물리 시스템의 변수는 두 가지로 분류된다.

| 구분 | Through Variable (관통 변수) | Across Variable (횡단 변수) |
|------|-----------------|-----------------|
| 정의 | 한 지점에서 측정 가능한 변수 | 양단의 **차이**를 측정하는 변수 |
| 특징 | 어느 지점에서 측정해도 동일 | 두 지점 간의 차이값 |
| 전기 | 전류 $i$ | 전압 $v$ |
| 기계 | 힘 $f$, 토크 $\tau$ | 속도 $v$, 각속도 $\omega$ |

**예시**

$$t_a - t_c = 0 \quad \text{(Through Variable, 관통 변수)}$$

$$\omega_a - \omega_c = \omega_t \quad \text{(Across Variable, 횡단 변수)}$$

---

## 2. 전기 시스템과 기계 시스템의 대응 🔑

전기 시스템과 기계 시스템은 수학적으로 동일한 형태를 가진다.

### 2-1. 인덕터 $L$ ↔ 스프링 강성의 역수 $\frac{1}{k}$

스프링 법칙 $f = kx$를 미분하면:

$$\frac{df}{dt} = k\frac{dx}{dt} \implies v = \frac{1}{k}\frac{df}{dt}$$

인덕터 방정식:

$$v = L\frac{di}{dt}$$

- $df$, $di$ 모두 **Through Variable (관통 변수)**
- 따라서 $L \longleftrightarrow \dfrac{1}{k}$

---

### 2-2. 커패시터 $C$ ↔ 질량 $M$

$$i = C\frac{dv}{dt} \qquad \longleftrightarrow \qquad f = M\frac{dv}{dt}$$

- 따라서 $C \longleftrightarrow M$

---

### 2-3. 저항 $R$ ↔ 감쇠계수 $b$

$$i = \frac{1}{R}v \qquad \longleftrightarrow \qquad f = bv$$

- 둘 다 **에너지를 소모**하는 소자
  - $R$ : 전기장 방해
  - $b$ : 속도 방해
- 따라서 $\dfrac{1}{R} \longleftrightarrow b$, 즉 $R \longleftrightarrow \dfrac{1}{b}$

---

### 2-4. 대응 관계 총정리

| 기계 시스템 | 전기 시스템 |
|:-----------:|:-----------:|
| $f$ (힘) | $i$ (전류) |
| $v$ (속도) | $v$ (전압) |
| $M$ (질량) | $C$ (커패시터) |
| $\dfrac{1}{k}$ (스프링) | $L$ (인덕터) |
| $b$ (감쇠) | $\dfrac{1}{R}$ (저항) |

> 💡 기계에서의 **벽(wall)** = 전기에서의 **GND** (기준점, 값 = 0)

## 3. 기계 시스템의 미분방정식

### 기본 공식

$$f = ma, \quad f = bv, \quad f = kx$$

외부에서 가해지는 힘 = 각 소자에 가해지는 힘의 합

$$\boxed{f(t) = m\ddot{x}(t) + b\dot{x}(t) + kx(t)}$$

### 미분방정식 차수

| 상태변수 | 차수 |
|---------|------|
| 속도 $v$ | 1차 미분방정식 |
| 위치 $x$ | 2차 미분방정식 |

---

### ✏️ 문제 풀이 절차 (기계 시스템)

1. **질량을 중심으로 변수 설정**
   - 질량의 수 = 변수의 수 = 1차 미분방정식의 수

2. **각 질량에 대해 운동방정식 수립**
   - 해당 질량에 **직접 가해지는 힘**만 고려 (전달되는 힘 제외)

**예시 (질량 2개)**

$$f(t) = m_1\ddot{y}_1 + k_1(y_1 - y_2) + k_2 y_1 \quad \cdots \text{질량 1 방정식}$$

$$0 = m_2\ddot{y}_2 + k_1(y_2 - y_1) + b\dot{y}_2 \quad \cdots \text{질량 2 방정식}$$

---

## 4. 전기 회로의 미분방정식

에너지 저장 소자 $L$, $C$의 수 = **미분방정식의 차수**

$$\frac{1}{R}v'(t) + C\ddot{v}(t) + \frac{1}{L}v(t) = r'(t)$$

## 5. 연속 LTI 시스템 (Linear Time-Invariant, 선형 시불변 시스템)

선형 시불변 시스템 — 제어공학에서 다루는 **이상적인 시스템**

$$\sum_k a_k \frac{d^k}{dt^k}y(t) = \sum_k b_k \frac{d^k}{dt^k}x(t)$$

### 완전해 (Complete Solution, 완전 해)

$$y(t) = \underbrace{y^{(h)}(t)}_{\text{동차해 (Homogeneous Solution)}} + \underbrace{y^{(p)}(t)}_{\text{특수해 (Particular Solution)}}$$

| 구분 | 의미 |
|------|------|
| $y^{(h)}$ : 동차해 (Homogeneous Solution) | 외부 입력 = 0, 초기조건에 의한 응답 **(자연응답, Natural Response)** |
| $y^{(p)}$ : 특수해 (Particular Solution) | 외부 입력에 의한 응답 **(강제응답, Forced Response)** |

### 특성방정식 구하기

일반해 $y = Ce^{st}$를 미분방정식에 대입 → **특성방정식** 도출

**예시**

$$\frac{dy(t)}{dt} + y(t) = r(t)$$

특성방정식: $s + 1 = 0 \implies s = -1$

$$y^{(h)} = Ce^{-t}$$

### 시스템 안정성 판별

| 조건 | 안정성 |
|------|--------|
| $\text{Re}(s) < 0$ | ✅ 안정 (시간이 지나면 0으로 수렴) |
| $\text{Re}(s) > 0$ | ❌ 불안정 (시간이 지나면 무한대로 발산) |

## 6. 라플라스 변환 🔑

> 💡 **라플라스 변환은 제어공학의 핵심 도구 — 반드시 암기할 것**

### 주요 변환 쌍

| 시간 영역 $f(t)$ | 라플라스 영역 $F(s)$ |
|:----------------:|:-------------------:|
| $\delta(t)$ (임펄스, 충격 함수) | $1$ |
| $u(t)$ (단위계단 함수, Unit Step) | $\dfrac{1}{s}$ |
| $e^{-at}$ | $\dfrac{1}{s+a}$ |
| $\sin\omega t$ | $\dfrac{\omega}{s^2+\omega^2}$ |
| $\cos\omega t$ | $\dfrac{s}{s^2+\omega^2}$ |
| $t^n$ | $\dfrac{n!}{s^{n+1}}$ |
| $e^{-at}\sin\omega t$ | $\dfrac{\omega}{(s+a)^2+\omega^2}$ |
| $e^{-at}\cos\omega t$ | $\dfrac{s+a}{(s+a)^2+\omega^2}$ |

### 미분·적분의 라플라스 변환

$$\mathcal{L} \left( \frac{d}{dt}x(t) \right) = sX(s) - x(0)$$

$$\mathcal{L} \left( \frac{d^k}{dt^k}x(t) \right) = s^k X(s) - s^{k-1}x(0) - s^{k-2}x'(0) - \cdots - x^{(k-1)}(0)$$

$$\mathcal{L} \left( \frac{d^2}{dt^2}x(t) \right) = s^2 X(s) - s x(0) - x'(0)$$

$$\mathcal{L} \left( \int x(t)\,dt \right) = \frac{X(s)}{s}$$

### 전달함수 (Transfer Function)

$$\boxed{H(s) = \frac{Y(s)}{X(s)}}$$

- 라플라스 변환 후 **분모 = 0** 으로 놓은 것이 **특성방정식**
- 임펄스(충격) 입력 $x(t) = \delta(t)$을 인가하면 $X(s) = 1$이므로:

$$H(s) = Y(s)$$

즉, **임펄스(충격) 응답의 라플라스 변환 = 전달함수**
