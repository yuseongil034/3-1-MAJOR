# PART 3. 벡터함수의 미적분

---

## 1. 벡터함수의 도함수와 적분

### 1-1. 도함수 (Derivative)

벡터함수 $\vec{r}(t) = \langle f(t),\, g(t),\, h(t) \rangle$의 도함수:

$$\boxed{\vec{r}\,'(t) = \langle f'(t),\, g'(t),\, h'(t) \rangle}$$

> 💡 **핵심 요령**: 벡터함수의 미분도 각 성분별로 따로 미분하면 된다.

**기하학적 의미**
- $\vec{r}\,'(t)$: 점 $\vec{r}(t)$에서의 **접선벡터(Tangent Vector)**
- 크기 $|\vec{r}\,'(t)|$: 그 점에서의 **속력(Speed)**

**단위접선벡터 (Unit Tangent Vector):**

$$\boxed{\vec{T}(t) = \frac{\vec{r}\,'(t)}{|\vec{r}\,'(t)|}}$$

---

### 1-2. ⭐ 벡터함수 미분 법칙

스칼라 함수와 동일한 규칙이 적용되지만, **외적의 경우 순서를 바꾸면 안 된다!**

| 법칙 | 공식 |
|------|------|
| 합의 법칙 | $\dfrac{d}{dt}[\vec{u}+\vec{v}] = \vec{u}' + \vec{v}'$ |
| 스칼라 곱 | $\dfrac{d}{dt}[f(t)\vec{u}] = f'(t)\vec{u} + f(t)\vec{u}'$ |
| 내적 | $\dfrac{d}{dt}[\vec{u}\cdot\vec{v}] = \vec{u}'\cdot\vec{v} + \vec{u}\cdot\vec{v}'$ |
| 외적 | $\dfrac{d}{dt}[\vec{u}\times\vec{v}] = \vec{u}'\times\vec{v} + \vec{u}\times\vec{v}'$ |
| 합성함수 | $\dfrac{d}{dt}[\vec{u}(f(t))] = f'(t)\,\vec{u}'(f(t))$ |

> ⚠️🚨 **외적 미분 시 순서 절대 바꾸지 말 것!** $\vec{u}'\times\vec{v} \neq \vec{v}\times\vec{u}'$

---

### 1-3. ⭐🔑 단위벡터의 미분 — 중요 성질

$|\vec{r}(t)| = c$ (상수)이면:

$$\boxed{\vec{r}(t) \cdot \vec{r}\,'(t) = 0}$$

> 💡 **직관**: 크기가 일정한 벡터(예: 단위벡터)는 자신의 미분과 항상 **수직**이다.  
> 🧮 **증명**: $|\vec{r}|^2 = c^2$ 양변 미분 → $2\vec{r}\cdot\vec{r}' = 0$

---

### 1-4. 적분 (Integral)

$$\boxed{\int \vec{r}(t)\, dt = \left\langle \int f(t)\,dt,\; \int g(t)\,dt,\; \int h(t)\,dt \right\rangle + \vec{C}}$$

> 💡 적분도 각 성분별로 따로 적분하면 된다. $\vec{C}$는 상수벡터.

**정적분:**

$$\int_a^b \vec{r}(t)\, dt = \left\langle \int_a^b f(t)\,dt,\; \int_a^b g(t)\,dt,\; \int_a^b h(t)\,dt \right\rangle$$

## 2. 곡선의 길이와 곡률

### 2-1. 매개변수 곡선의 길이 함수

#### 🎯 호의 길이 (Arc Length) ⭐

매개변수 $t \in [a, b]$로 표현된 3차원 곡선의 전체 길이:

$$\boxed{L = \int_a^b |\vec{r}\,'(t)|\, dt = \int_a^b \sqrt{[f'(t)]^2 + [g'(t)]^2 + [h'(t)]^2}\; dt}$$

2차원 곡선 $x=f(t)$, $y=g(t)$의 경우:

$$L = \int_a^b \sqrt{[f'(t)]^2 + [g'(t)]^2}\; dt$$

---

#### 🔑📌 길이 함수 (Arc Length Function)

고정점 $t = a$부터 $t$까지의 **누적 길이**:

$$\boxed{s(t) = \int_a^t |\vec{r}\,'(u)|\, du}$$

미분하면:

$$\frac{ds}{dt} = |\vec{r}\,'(t)|$$

> 💡 **언제 사용하나?**  
> 곡선을 호의 길이 $s$로 **재매개변수화** 할 때 사용한다.  
> $s$로 매개변수화하면 $|\vec{T}| = 1$이 보장되어 곡률 계산이 단순해진다.

**호의 길이로 재매개변수화 절차:**

```
① s(t) = ∫|r'(u)|du 를 계산한다
② t를 s의 함수로 역으로 표현: t = t(s)
③ r(t(s))에 대입하면 호의 길이로 매개변수화 완료
```

---

### 2-2. 🎯⭐ 곡률 (Curvature)

곡선이 **얼마나 휘어있는지**를 나타내는 값.

$$\boxed{\kappa = \left|\frac{d\vec{T}}{ds}\right| = \frac{|\vec{T}'(t)|}{|\vec{r}'(t)|}}$$

**🔑 계산하기 쉬운 공식 (시험에서 주로 사용):**

$$\boxed{\kappa = \frac{|\vec{r}'(t) \times \vec{r}''(t)|}{|\vec{r}'(t)|^3}}$$

> 💡 **직관적 이해**
> - $\kappa$가 크다 → 곡선이 급하게 꺾임 (예: 급커브)
> - $\kappa = 0$ → 직선
> - $\kappa$가 일정 → 원

**2차원 $y = f(x)$ 곡선의 곡률:**

$$\kappa = \frac{|f''(x)|}{\left[1 + (f'(x))^2\right]^{3/2}}$$

**곡률 반지름 (Radius of Curvature):**

$$\boxed{\rho = \frac{1}{\kappa}}$$

- 📌 곡률이 클수록 반지름은 작아짐 (더 급하게 꺾임)

---

### 2-3. 🔑 법선벡터와 이중법선벡터

#### 🎯 주법선벡터 (Principal Normal Vector) $\vec{N}$

접선벡터 $\vec{T}$의 변화 방향 → **곡선이 휘어지는 방향**을 가리킴.

$$\boxed{\vec{N}(t) = \frac{\vec{T}'(t)}{|\vec{T}'(t)|}}$$

- ✅ $\vec{N}$은 항상 $\vec{T}$와 **수직**
- ✅ 곡선이 꺾이는 **안쪽 방향**을 가리킴

#### 이중법선벡터 (Binormal Vector) $\vec{B}$

$\vec{T}$와 $\vec{N}$ 모두에 수직인 벡터.

$$\boxed{\vec{B}(t) = \vec{T}(t) \times \vec{N}(t)}$$

- ✅ $|\vec{B}| = 1$ (단위벡터)
- ✅ $\vec{T}$, $\vec{N}$, $\vec{B}$는 서로 **수직인 단위벡터 3개** → **Frenet 틀(Frame)** 형성

#### 🎯⭐ TNB 틀 (Frenet-Serret Frame) 요약

```
T: 접선 방향 (곡선이 나아가는 방향)
N: 주법선 방향 (곡선이 꺾이는 방향, T에 수직)
B: 이중법선 방향 (T와 N에 모두 수직)

세 벡터는 항상 서로 수직 → 3차원 좌표축처럼 작동
```

| 벡터 | 공식 | 방향 |
|------|------|------|
| $\vec{T}$ | $\vec{r}'/ \|\vec{r}'\|$ | 진행 방향 |
| $\vec{N}$ | $\vec{T}'/\|\vec{T}'\|$ | 꺾이는 방향 (안쪽) |
| $\vec{B}$ | $\vec{T} \times \vec{N}$ | 두 방향에 수직 |

## 3. 법평면, 접촉평면, 접촉원

### 3-1. 🔑 법평면 (Normal Plane)

점 $P$에서 **접선벡터 $\vec{T}$에 수직**인 평면.  
법평면의 법선벡터 = $\vec{r}'(t)$ (접선벡터)

$$\boxed{f'(t_0)(x - x_0) + g'(t_0)(y - y_0) + h'(t_0)(z - z_0) = 0}$$

> 💡 **언제**: 곡선 위의 점에서 진행 방향에 수직인 단면을 구할 때

---

### 3-2. 🔑 접촉평면 (Osculating Plane)

점 $P$에서 **$\vec{T}$와 $\vec{N}$을 모두 포함**하는 평면.  
접촉평면의 법선벡터 = $\vec{B}$ (이중법선벡터)

$$\vec{B}(t_0) \cdot \langle x - x_0,\, y - y_0,\, z - z_0 \rangle = 0$$

> 💡 **직관**: 곡선이 "가장 많이 닿는" 평면. 곡선이 꺾이는 평면이라고 생각하면 됨.

---

### 3-3. 접촉원 (Osculating Circle)

점 $P$에서 곡선에 **가장 잘 맞는 원**.

- 반지름 = 곡률 반지름 $\rho = \dfrac{1}{\kappa}$
- 중심 = 곡률 중심 (곡선의 오목한 방향, $\vec{N}$ 방향)
- 접촉평면 위에 놓임

$$\boxed{r = \frac{1}{\kappa}}$$

> 💡 **직관**: 곡선을 그 점에서 원으로 근사하는 것. 곡선이 급할수록 원이 작아짐.

---

### 3-4. ✏️📋 법평면·접촉평면 구하는 절차

```
① r'(t₀), r''(t₀) 계산
② T = r'/|r'|, N = T'/|T'|, B = T × N 계산
③ 법평면: 법선벡터 = r'(t₀) → 평면 방정식 수립
④ 접촉평면: 법선벡터 = B(t₀) → 평면 방정식 수립
```

## 4. 매개변수 곡면과 매개변수 방정식

### 4-1. 🔑 매개변수 곡면 (Parametric Surface)

두 매개변수 $u$, $v$로 표현되는 3차원 곡면.

$$\boxed{\vec{r}(u, v) = \langle x(u,v),\; y(u,v),\; z(u,v) \rangle}$$

- 곡선: 매개변수 **1개** ($t$)
- 곡면: 매개변수 **2개** ($u$, $v$)

**접선벡터 (편미분으로 구함):**

$$\vec{r}_u = \frac{\partial \vec{r}}{\partial u} = \left\langle \frac{\partial x}{\partial u},\, \frac{\partial y}{\partial u},\, \frac{\partial z}{\partial u} \right\rangle$$

$$\vec{r}_v = \frac{\partial \vec{r}}{\partial v} = \left\langle \frac{\partial x}{\partial v},\, \frac{\partial y}{\partial v},\, \frac{\partial z}{\partial v} \right\rangle$$

**🎯 곡면의 법선벡터:**

$$\boxed{\vec{n} = \vec{r}_u \times \vec{r}_v}$$

> ⚠️🚨 $\vec{r}_u \times \vec{r}_v \neq \vec{r}_v \times \vec{r}_u$ — 순서에 따라 방향이 반대!

**곡면 넓이 요소:**

$$\boxed{dS = |\vec{r}_u \times \vec{r}_v|\, du\, dv}$$

---

### 4-2. 🎯⭐ 구의 매개변수 방정식

반지름 $a$인 구 $x^2 + y^2 + z^2 = a^2$:

$$\boxed{\vec{r}(\phi, \theta) = \langle a\sin\phi\cos\theta,\; a\sin\phi\sin\theta,\; a\cos\phi \rangle}$$

| 매개변수 | 의미 | 범위 |
|----------|------|------|
| $\phi$ (천정각) | $z$축과 이루는 각 | $0 \leq \phi \leq \pi$ |
| $\theta$ (방위각) | $xy$평면에서의 각 | $0 \leq \theta \leq 2\pi$ |

**구의 법선벡터 및 넓이 요소:**

$$|\vec{r}_\phi \times \vec{r}_\theta| = a^2\sin\phi$$

$$\boxed{dS = a^2\sin\phi\; d\phi\; d\theta}$$

**✅ 구의 전체 넓이 검산:**

$$S = \int_0^{2\pi}\int_0^{\pi} a^2\sin\phi\; d\phi\; d\theta = 4\pi a^2 \checkmark$$

---

### 4-3. 🎯⭐ 원기둥의 매개변수 방정식

반지름 $a$, 높이 $h$인 원기둥 $x^2 + y^2 = a^2$:

$$\boxed{\vec{r}(\theta, z) = \langle a\cos\theta,\; a\sin\theta,\; z \rangle}$$

| 매개변수 | 의미 | 범위 |
|----------|------|------|
| $\theta$ | 회전각 | $0 \leq \theta \leq 2\pi$ |
| $z$ | 높이 | $0 \leq z \leq h$ |

**원기둥의 넓이 요소:**

$$\vec{r}_\theta = \langle -a\sin\theta,\; a\cos\theta,\; 0 \rangle, \quad \vec{r}_z = \langle 0,\; 0,\; 1 \rangle$$

$$|\vec{r}_\theta \times \vec{r}_z| = a$$

$$\boxed{dS = a\; d\theta\; dz}$$

> 💡 ✅ 원기둥 옆면 넓이 검산: $S = \int_0^{2\pi}\int_0^h a\; dz\; d\theta = 2\pi a h \checkmark$

---

### 4-4. 🔑 회전 곡면 (Surface of Revolution)

$xz$평면의 곡선 $z = f(x)$ ($a \leq x \leq b$)를 **$z$축을 중심으로 회전**시킨 곡면:

$$\boxed{\vec{r}(x, \theta) = \langle x\cos\theta,\; x\sin\theta,\; f(x) \rangle}$$

| 매개변수 | 의미 | 범위 |
|----------|------|------|
| $x$ | 원래 곡선의 $x$값 | $a \leq x \leq b$ |
| $\theta$ | 회전각 | $0 \leq \theta \leq 2\pi$ |

**넓이 요소:**

$$\vec{r}_x = \langle \cos\theta,\; \sin\theta,\; f'(x) \rangle, \quad \vec{r}_\theta = \langle -x\sin\theta,\; x\cos\theta,\; 0 \rangle$$

$$|\vec{r}_x \times \vec{r}_\theta| = x\sqrt{1 + [f'(x)]^2}$$

$$\boxed{dS = x\sqrt{1 + [f'(x)]^2}\; dx\; d\theta}$$

**🎯 회전 곡면의 넓이:**

$$\boxed{S = 2\pi \int_a^b x\sqrt{1 + [f'(x)]^2}\; dx}$$

> ⚠️📌 적분 앞의 $2\pi$는 $\theta$를 $0$에서 $2\pi$까지 적분한 결과임!

---

### 4-5. ✏️📋 매개변수 곡면 종류별 정리

| 곡면 | 매개변수 방정식 | 넓이 요소 $dS$ |
|------|---------------|---------------|
| 구 (반지름 $a$) | $\langle a\sin\phi\cos\theta,\, a\sin\phi\sin\theta,\, a\cos\phi \rangle$ | $a^2\sin\phi\; d\phi\; d\theta$ |
| 원기둥 (반지름 $a$) | $\langle a\cos\theta,\, a\sin\theta,\, z \rangle$ | $a\; d\theta\; dz$ |
| 회전 곡면 | $\langle x\cos\theta,\, x\sin\theta,\, f(x) \rangle$ | $x\sqrt{1+(f')^2}\; dx\; d\theta$ |
| 일반 곡면 $z=g(x,y)$ | $\langle x,\, y,\, g(x,y) \rangle$ | $\sqrt{g_x^2 + g_y^2 + 1}\; dA$ |

## 📋⭐ 전체 요약 — 공식 한눈에 보기 (시험 직전 확인용)

### 🔑 벡터함수 미적분 요약

| 항목 | 공식 |
|------|------|
| 도함수 | $\vec{r}'(t) = \langle f',\, g',\, h' \rangle$ |
| 단위접선벡터 | $\vec{T} = \vec{r}'/\|\vec{r}'\|$ |
| 적분 | $\int \vec{r}\,dt = \langle \int f,\, \int g,\, \int h \rangle + \vec{C}$ |
| 단위벡터 미분 성질 | $\vec{r}\cdot\vec{r}' = 0$ (크기 일정 시) |

### 🎯⭐ TNB 틀 요약

| 벡터 | 공식 |
|------|------|
| $\vec{T}$ (접선) | $\vec{r}'/\|\vec{r}'\|$ |
| $\vec{N}$ (주법선) | $\vec{T}'/\|\vec{T}'\|$ |
| $\vec{B}$ (이중법선) | $\vec{T}\times\vec{N}$ |

### 🎯⭐ 곡률 공식 요약

| 상황 | 공식 |
|------|------|
| ⭐ 일반 (가장 많이 사용) | $\kappa = \|\vec{r}'\times\vec{r}''\| / \|\vec{r}'\|^3$ |
| $y=f(x)$ | $\kappa = \|f''\| / (1+(f')^2)^{3/2}$ |
| 곡률 반지름 | $\rho = 1/\kappa$ |

### 🔑 평면 요약

| 평면 | 법선벡터 |
|------|---------|
| 법평면 (Normal Plane) | $\vec{T}$ (= $\vec{r}'$) |
| 접촉평면 (Osculating Plane) | $\vec{B}$ |

### 🎯 매개변수 곡면 법선벡터·넓이 요소

$$\vec{n} = \vec{r}_u \times \vec{r}_v, \qquad dS = |\vec{r}_u \times \vec{r}_v|\, du\, dv$$
