## 1. 극한과 연속

### 핵심 극한 공식

| 이름 | 공식 |
|------|------|
| 삼각함수 극한 | $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$ |
| 삼각함수 극한 2 | $\displaystyle\lim_{x \to 0} \frac{1 - \cos x}{x} = 0$ |
| 자연상수 정의 | $\displaystyle\lim_{n \to \infty}\left(1 + \frac{1}{n}\right)^{n} = e$ |
| 자연상수 (일반형) | $\displaystyle\lim_{x \to 0} (1+x)^{1/x} = e$ |

### 로피탈 정리 ⭐

$$\lim_{x\to a}\frac{f(x)}{g(x)} \overset{\left(\frac{0}{0}\;\text{or}\;\frac{\infty}{\infty}\right)}{=} \lim_{x\to a}\frac{f'(x)}{g'(x)}$$

> 💡 **EXAM TIP**  
> 로피탈 정리는 반드시 $\frac{0}{0}$ 또는 $\frac{\infty}{\infty}$ 꼴일 때만 적용 가능합니다.  
> 적용 전 부정형 여부를 반드시 확인하세요.  
> 전자공학에서는 $\frac{\sin\omega t}{\omega t}$ 형태(sinc 함수)를 자주 만납니다.

## 2. 미분 기본 공식

### 기본 미분 법칙

| 이름 | 공식 |
|------|------|
| 거듭제곱 법칙 | $\dfrac{d}{dx}x^n = nx^{n-1}$ |
| 합·차의 법칙 | $(f \pm g)' = f' \pm g'$ |
| 곱의 법칙 | $(fg)' = f'g + fg'$ |
| 몫의 법칙 | $\left(\dfrac{f}{g}\right)' = \dfrac{f'g - fg'}{g^2}$ |
| **연쇄 법칙 (Chain Rule) ★★★** | $\dfrac{d}{dx}f(g(x)) = f'(g(x))\cdot g'(x)$ |
| 역함수 미분 | $\dfrac{dx}{dy} = \dfrac{1}{dy/dx}$ |

### 미분의 정의

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

> 💡 **EXAM TIP**  
> Chain Rule은 신호·시스템, 회로 해석 등 전자공학 전반에서 핵심입니다.  
> $e^{j\omega t}$, $\sin(\omega t + \phi)$를 미분할 때 내부 함수의 미분값 $\omega$를 반드시 곱해야 합니다.

## 3. 적분 기본 공식

### 기본 부정적분

| 피적분함수 | 결과 |
|-----------|------|
| $x^n \; (n \neq -1)$ | $\dfrac{x^{n+1}}{n+1} + C$ |
| $\dfrac{1}{x}$ | $\ln\|x\| + C$ |
| $e^x$ | $e^x + C$ |
| $e^{ax}$ | $\dfrac{1}{a}e^{ax} + C$ |
| $a^x$ | $\dfrac{a^x}{\ln a} + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\tan x$ | $-\ln\|\cos x\| + C$ |
| $\sec^2 x$ | $\tan x + C$ |
| $\dfrac{1}{1+x^2}$ | $\arctan x + C$ |
| $\dfrac{1}{\sqrt{1-x^2}}$ | $\arcsin x + C$ |

### 미적분학의 기본정리

$$\int_a^b f(x)\,dx = F(b) - F(a) \qquad \text{단, } F'(x) = f(x)$$

> 💡 **EXAM TIP**  
> $\dfrac{d}{dx}\displaystyle\int_a^{x} f(t)\,dt = f(x)$ — 적분 상한이 변수일 때 이 공식을 씁니다.  
> 회로 이론에서 커패시터 전압 $v(t) = \dfrac{1}{C}\int i\,dt$를 다룰 때 핵심입니다.

## 4. 삼각함수 미·적분 & 항등식

### 삼각함수 미분

| 함수 | 미분 |
|------|------|
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $\arctan x$ | $\dfrac{1}{1+x^2}$ |

### 핵심 삼각 항등식 (ECE 필수)

| 항등식 | 공식 |
|--------|------|
| 피타고라스 | $\sin^2 x + \cos^2 x = 1$ |
| 배각 (sin) | $\sin 2x = 2\sin x\cos x$ |
| 배각 (cos) | $\cos 2x = \cos^2 x - \sin^2 x$ |
| 반각 (sin²) | $\sin^2 x = \dfrac{1-\cos 2x}{2}$ |
| 반각 (cos²) | $\cos^2 x = \dfrac{1+\cos 2x}{2}$ |
| **오일러 공식 ★★★** | $e^{j\theta} = \cos\theta + j\sin\theta$ |

> 💡 **ECE 핵심**  
> 반각 공식은 전력공학에서 $\sin^2(\omega t)$의 평균값 계산에 필수입니다.  
> 오일러 공식은 페이저(Phasor), 푸리에 변환, 라플라스 변환 등 전자공학 전반의 기초이므로 완벽히 암기하세요.

## 5. 지수·로그함수 미·적분

### 미분 공식

| 함수 | 미분 |
|------|------|
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\dfrac{1}{x}$ |
| $\log_a x$ | $\dfrac{1}{x\ln a}$ |
| $e^{st}$ (복소 지수) | $s\,e^{st}$ |

**로그 미분법 예시:** $y = x^x \;\Rightarrow\; \ln y = x\ln x$

### 로그 법칙 (계산 단순화)

$$\ln(ab) = \ln a + \ln b \qquad \ln\!\left(\frac{a}{b}\right) = \ln a - \ln b$$

$$\ln(a^n) = n\ln a \qquad \ln e = 1, \quad e^0 = 1$$

## 6. 적분 테크닉

### ① 치환 적분 (u-substitution)

$$\int f(g(x))\,g'(x)\,dx = \int f(u)\,du \qquad (u = g(x))$$

### ② 부분 적분 ⭐

$$\int u\,dv = uv - \int v\,du$$

> 💡 **LIATE 규칙** — $u$ 선택 우선순위  
> **L**og → **I**nverse trig → **A**lgebraic → **T**rig → **E**xponential  
> 예) $\int x e^x dx$ → $u = x$, $dv = e^x dx$  
> 신호 분석에서 $\int t\,e^{-st}dt$ 형태를 자주 계산합니다.

### ③ 부분분수 분해 (라플라스 역변환 핵심)

$$\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$$

### ④ 삼각치환

| 포함 형태 | 치환 | 사용 항등식 |
|-----------|------|------------|
| $\sqrt{a^2 - x^2}$ | $x = a\sin\theta$ | $1 - \sin^2\theta = \cos^2\theta$ |
| $\sqrt{a^2 + x^2}$ | $x = a\tan\theta$ | $1 + \tan^2\theta = \sec^2\theta$ |
| $\sqrt{x^2 - a^2}$ | $x = a\sec\theta$ | $\sec^2\theta - 1 = \tan^2\theta$ |

### ⑤ 자주 나오는 정적분 결과

| 이름 | 결과 |
|------|------|
| 가우스 적분 (ECE 필수) | $\displaystyle\int_{-\infty}^{\infty} e^{-x^2}dx = \sqrt{\pi}$ |
| 삼각함수 직교성 | $\displaystyle\int_0^{2\pi}\sin(nx)\sin(mx)\,dx = \pi\delta_{nm}$ |

## 7. 급수 & 테일러/맥클로린 전개

### 테일러 급수 일반형

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$$

### 맥클로린 전개 (a = 0) — ECE 필수 암기

| 함수 | 전개식 |
|------|--------|
| $e^x$ | $\displaystyle\sum_{n=0}^{\infty}\frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots$ |
| $\sin x$ | $\displaystyle\sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$ |
| $\cos x$ | $\displaystyle\sum_{n=0}^{\infty}\frac{(-1)^n x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots$ |
| $\ln(1+x)$ | $x - \dfrac{x^2}{2} + \dfrac{x^3}{3} - \cdots \quad (\|x\| \le 1)$ |
| $\dfrac{1}{1-x}$ | $1 + x + x^2 + x^3 + \cdots \quad (\|x\| < 1)$ |
| $(1+x)^n$ | $1 + nx + \dfrac{n(n-1)}{2!}x^2 + \cdots$ |

> 💡 **ECE 활용**  
> 소신호 근사(small signal approximation)에서 $e^{j\omega t} \approx 1 + j\omega t$, $\sin\theta \approx \theta$ ($\theta \ll 1$) 등 1차 테일러 근사가 자주 쓰입니다.  
> 이는 트랜지스터 소신호 모델의 기초입니다.

## 8. 편미분 & 다변수 미적분

### 편미분 기본

| 이름 | 공식 |
|------|------|
| 편미분 정의 | $\dfrac{\partial f}{\partial x} = \displaystyle\lim_{h\to 0}\dfrac{f(x+h,y)-f(x,y)}{h}$ |
| 클레로 정리 | $\dfrac{\partial^2 f}{\partial x\,\partial y} = \dfrac{\partial^2 f}{\partial y\,\partial x}$ |
| 전미분 | $df = \dfrac{\partial f}{\partial x}dx + \dfrac{\partial f}{\partial y}dy$ |
| 연쇄 법칙 (다변수) | $\dfrac{dz}{dt} = \dfrac{\partial z}{\partial x}\dfrac{dx}{dt} + \dfrac{\partial z}{\partial y}\dfrac{dy}{dt}$ |

### 다중 적분

$$\iint_D f(x,y)\,dA = \int_a^b\!\int_{g_1(x)}^{g_2(x)}\! f(x,y)\,dy\,dx$$

### 야코비안 (좌표 변환 — 원통 좌표)

$$\iint f(x,y)\,dx\,dy = \iint f(r,\theta)\underbrace{\,r\,}_{J}\,dr\,d\theta$$

> 💡 **ECE 활용**  
> 전자기학에서 가우스 법칙, 패러데이 법칙 등을 면적분·선적분으로 표현할 때 다변수 미적분이 필수입니다.  
> 원통·구면 좌표계의 야코비안을 반드시 익혀두세요.

## 9. 벡터 미적분 기초

> **⚡ 전자기학 필수 파트**

### 벡터 미분 연산자

| 연산자 | 정의 |
|--------|------|
| 그래디언트 (Gradient) | $\nabla f = \dfrac{\partial f}{\partial x}\hat{x} + \dfrac{\partial f}{\partial y}\hat{y} + \dfrac{\partial f}{\partial z}\hat{z}$ |
| 발산 (Divergence) | $\nabla\cdot\mathbf{F} = \dfrac{\partial F_x}{\partial x}+\dfrac{\partial F_y}{\partial y}+\dfrac{\partial F_z}{\partial z}$ |
| 회전 (Curl) | $\nabla\times\mathbf{F} = \det\begin{pmatrix}\hat{x}&\hat{y}&\hat{z}\\\partial_x&\partial_y&\partial_z\\F_x&F_y&F_z\end{pmatrix}$ |
| 라플라시안 | $\nabla^2 f = \dfrac{\partial^2 f}{\partial x^2}+\dfrac{\partial^2 f}{\partial y^2}+\dfrac{\partial^2 f}{\partial z^2}$ |

### 적분 정리

| 정리 | 공식 |
|------|------|
| 발산 정리 (Gauss) | $\displaystyle\oiint_S \mathbf{F}\cdot d\mathbf{A} = \iiint_V (\nabla\cdot\mathbf{F})\,dV$ |
| 스토크스 정리 | $\displaystyle\oint_C \mathbf{F}\cdot d\mathbf{l} = \iint_S (\nabla\times\mathbf{F})\cdot d\mathbf{A}$ |

> 💡 **맥스웰 방정식 연결**  
> 발산 정리 → 가우스 법칙  
> 스토크스 정리 → 패러데이·암페어 법칙  
> 전자기학 수업 전에 이 두 정리를 완벽히 이해해두면 맥스웰 방정식이 훨씬 쉬워집니다.

> ⚠️ **주의 — 벡터 항등식**  
> $\nabla \times (\nabla f) = 0$ (그래디언트의 컬은 항상 0)  
> $\nabla \cdot (\nabla \times \mathbf{F}) = 0$ (컬의 발산은 항상 0)  
> 이 두 항등식은 전자기학 증명에서 자주 활용됩니다.
