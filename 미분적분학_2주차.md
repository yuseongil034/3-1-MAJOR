## 1. 이변수 함수 (Function of Two Variables)

### 개념

입력값이 **두 개($x$, $y$)** 이고 출력값이 **하나($z$)** 인 함수.

$$z = f(x,\, y)$$

- 일변수 함수 $y = f(x)$가 평면 위의 곡선을 그린다면,
- 이변수 함수 $z = f(x, y)$는 **3차원 공간 위의 곡면**을 그린다.

### 🗺️ 직관적 이해

> 지형도를 생각하면 쉽다.
> $x$, $y$는 지도의 위·경도, $z$는 그 지점의 **고도(높이)** 다.
> 이변수 함수는 지형의 높낮이를 수식으로 표현한 것이다.

### 정의역과 치역

- **정의역 (Domain)**: $f(x, y)$가 정의되는 $(x, y)$의 집합
- **치역 (Range)**: $f$가 출력하는 $z$값의 집합

**⚠️ 정의역 구하는 요령 — 시험 단골**

| 상황 | 제약 조건 |
|------|-----------|
| 분모에 식이 있을 때 | 분모 $\neq 0$ |
| 루트($\sqrt{\phantom{x}}$) 안에 식이 있을 때 | 루트 안 $\geq 0$ |
| $\ln$ 안에 식이 있을 때 | $\ln$ 안 $> 0$ |

**예시**

$$f(x, y) = \sqrt{1 - x^2 - y^2}$$

- 루트 안 $\geq 0$ → $1 - x^2 - y^2 \geq 0$ → $x^2 + y^2 \leq 1$
- 정의역: 반지름 1인 원의 내부 (원판)

### 등위곡선 (Level Curve) 💡

$z = f(x, y)$에서 $z = k$ (상수)로 놓았을 때 나타나는 $xy$평면 위의 곡선.

$$f(x, y) = k$$

- 지형도의 **등고선**과 동일한 개념
- 등위곡선들이 촘촘할수록 → 경사가 가파름
- 등위곡선들이 넓을수록 → 경사가 완만함

## 2. 선형함수와 평면 (Linear Function & Plane)

### 개념

이변수 함수 중 가장 단순한 형태. $z$가 $x$, $y$의 **일차식**으로 표현된다.

$$z = ax + by + c \quad \text{또는} \quad Ax + By + Cz = D$$

- 그래프는 **평면(Plane)** 이 된다.
- 계수 $a$, $b$는 각각 $x$, $y$ 방향으로의 기울기를 나타낸다.

### ✏️ 평면의 방정식 구하는 요령

**🔑 방법 1 — 점과 법선벡터를 알 때 (가장 자주 출제)**

평면 위의 점 $(x_0, y_0, z_0)$과 법선벡터 $\vec{n} = (A, B, C)$가 주어지면:

$$\boxed{A(x - x_0) + B(y - y_0) + C(z - z_0) = 0}$$

**방법 2 — 세 점을 알 때**

세 점 $P_1$, $P_2$, $P_3$이 주어지면:

1. 두 벡터 $\vec{v_1} = P_2 - P_1$, $\vec{v_2} = P_3 - P_1$ 계산
2. 법선벡터 $\vec{n} = \vec{v_1} \times \vec{v_2}$ (외적)
3. 방법 1 적용

> 💡 **언제 사용하나?**
> - 3차원 공간에서 평평한 표면을 표현할 때
> - 곡면의 접평면(Tangent Plane)을 구할 때
> - 두 평면이 이루는 각도를 구할 때

## 3. 포물주면 (Parabolic Cylinder)

### 개념

변수 **하나가 없는** 포물선 방정식 → 3차원에서 **기둥(주면)** 형태가 된다.

$$z = x^2 \quad \text{(}y\text{가 없음)}$$

- $xy$평면에서 $z = x^2$은 포물선
- 3차원에서는 이 포물선이 $y$축 방향으로 **무한히 늘어난 모양**

### 🔑 주면 판별 요령 — 변수 개수를 세어라!

> 📌 방정식에서 **변수가 하나 빠져 있으면** → 그 방향으로 늘어난 **주면(Cylinder)** 이다.

| 방정식 | 빠진 변수 | 도형 |
|--------|-----------|------|
| $z = x^2$ | $y$ | $y$축 방향 포물주면 |
| $z = y^2$ | $x$ | $x$축 방향 포물주면 |
| $x^2 + y^2 = r^2$ | $z$ | $z$축 방향 원기둥 |
| $x^2 + z^2 = r^2$ | $y$ | $y$축 방향 원기둥 |

### 직관적 이해

> 반원형 지붕(아치형 터널)을 떠올리면 된다.
> 단면이 포물선 모양이고, 그것이 한 방향으로 쭉 늘어난 형태.

## 4. 이차곡면 (Quadric Surface)

### 개념

$x$, $y$, $z$의 **이차방정식**으로 표현되는 3차원 곡면의 총칭.

$$Ax^2 + By^2 + Cz^2 + Dxy + Eyz + Fxz + Gx + Hy + Iz + J = 0$$

### 🔑 이차곡면 종류 총정리

#### (1) 타원면 (Ellipsoid) 🌐

$$\boxed{\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1}$$

- $a = b = c$이면 → **구(Sphere)**
- 모든 절단면이 타원 (또는 원)
- **언제**: 3차원에서 완전히 닫힌 달걀·공 모양 곡면

#### (2) 🔑 타원포물면 (Elliptic Paraboloid) — 시험 자주 출제

$$\boxed{z = \frac{x^2}{a^2} + \frac{y^2}{b^2}}$$

- 꼭짓점이 원점, 위쪽으로 열린 그릇 모양
- $z = k$ 절단면: **타원** (또는 원)
- $x = 0$ 또는 $y = 0$ 절단면: **포물선**
- **언제**: 위성 안테나, 반사경, 최솟값 문제

#### (3) 🔑 쌍곡포물면 (Hyperbolic Paraboloid) — 안장면, 헷갈리기 쉬운 형태

$$\boxed{z = \frac{x^2}{a^2} - \frac{y^2}{b^2}}$$

- **말안장(Saddle)** 모양
- $x$ 방향으로는 아래로 볼록, $y$ 방향으로는 위로 볼록
- ⚠️ 타원포물면과 **부호 하나 차이** — $-$ 부호 주의!
- **언제**: 안장점(Saddle Point) 문제, 최대·최소 판별

#### (4) 타원뿔면 (Elliptic Cone)

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = \frac{z^2}{c^2}$$

- 꼭짓점이 원점인 이중 원뿔 모양
- **언제**: 원뿔 모양 도형의 곡면 표현

#### (5) 일엽쌍곡면 (Hyperboloid of One Sheet)

$$\boxed{\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1}$$

- 허리가 잘록한 탑 모양 (냉각탑)
- ⚠️ **$z^2$ 앞에 $(-)$, 우변 $= 1$** ← 부호 암기!

#### (6) 이엽쌍곡면 (Hyperboloid of Two Sheets)

$$\boxed{-\frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1}$$

- 위아래 두 개의 분리된 곡면
- ⚠️ **$z^2$ 앞에 $(+)$, 나머지 $(-)$, 우변 $= 1$** ← 부호 암기!

### 🔑 이차곡면 판별 절차 — 이것만 외워라!

```
① 우변을 0 또는 1로 정리한다
② 각 항의 부호(+/-)를 확인한다
③ 아래 판별표를 적용한다
```

| 형태 | 판별 기준 |
|------|-----------|
| $+\,+\,+ = 1$ | 타원면 |
| $z = +\,+$ | 타원포물면 (위로 볼록) |
| $z = +\,-$ | 쌍곡포물면 (안장) |
| $+\,+\,- = 0$ | 타원뿔면 |
| $+\,+\,- = 1$ | 일엽쌍곡면 |
| $-\,-\,+ = 1$ | 이엽쌍곡면 |

## 5. 원주좌표와 구면좌표

### 5-1. 원주좌표 (Cylindrical Coordinates)

#### 개념

**극좌표 + $z$축** 을 합친 3차원 좌표계.

$$P = (r,\, \theta,\, z)$$

#### 직교좌표 ↔ 원주좌표 변환 🔑

$$\boxed{x = r\cos\theta, \quad y = r\sin\theta, \quad z = z}$$

$$\boxed{r = \sqrt{x^2 + y^2}, \quad \theta = \arctan\!\left(\frac{y}{x}\right), \quad z = z}$$

$$r^2 = x^2 + y^2$$

#### ✏️ 언제 원주좌표를 쓰나?

> 도형이나 적분 범위가 **$z$축에 대한 회전 대칭** 구조일 때 사용한다.

| 상황 | 예시 |
|------|------|
| 원기둥 내부 적분 | $x^2 + y^2 \leq a^2$ |
| 원뿔 내부 적분 | $z = \sqrt{x^2 + y^2}$ |
| 회전체 부피 계산 | 원통, 원뿔, 포물면 회전체 |

**⚠️ 적분 부피 요소 (야코비안 — 절대 빠뜨리면 안 됨!):**

$$\boxed{dV = r\, dr\, d\theta\, dz}$$

> ⚠️ 원주좌표로 적분 시 반드시 $r$을 곱해야 한다! $r$을 빠뜨리면 완전히 틀린 답이 나온다.

---

### 5-2. 구면좌표 (Spherical Coordinates)

#### 개념

원점으로부터의 **거리 $\rho$**, 천정각 $\phi$, 방위각 $\theta$로 위치를 나타내는 좌표계.

$$P = (\rho,\, \phi,\, \theta)$$

| 기호 | 의미 | 범위 |
|------|------|------|
| $\rho$ | 원점까지의 거리 | $\rho \geq 0$ |
| $\phi$ | $z$축과 이루는 각 (천정각) | $0 \leq \phi \leq \pi$ |
| $\theta$ | $xy$평면에서의 각 (방위각) | $0 \leq \theta \leq 2\pi$ |

#### 직교좌표 ↔ 구면좌표 변환 🔑

$$\boxed{x = \rho\sin\phi\cos\theta, \quad y = \rho\sin\phi\sin\theta, \quad z = \rho\cos\phi}$$

$$\boxed{\rho = \sqrt{x^2 + y^2 + z^2}}$$

**⚠️ 적분 부피 요소 (야코비안 — 절대 빠뜨리면 안 됨!):**

$$\boxed{dV = \rho^2 \sin\phi\, d\rho\, d\phi\, d\theta}$$

> ⚠️ 구면좌표 적분 시 $\rho^2 \sin\phi$를 반드시 곱해야 한다! 빠뜨리면 완전히 틀린 답이 나온다.

#### ✏️ 언제 구면좌표를 쓰나?

> 도형이 **원점을 중심으로 한 구(球) 형태**일 때 가장 강력하다.

| 상황 | 예시 |
|------|------|
| 구의 내부/외부 적분 | $x^2 + y^2 + z^2 \leq a^2$ |
| 구의 껍질 적분 | $a^2 \leq x^2 + y^2 + z^2 \leq b^2$ |
| 반구 부피·무게중심 | 반구, 구의 일부 |

### 🔑 좌표계 선택 요령

```
도형에 z축이 보이는가?
    ↓ YES
z축에 대칭인가?
    ├─ YES → 원주좌표 (r, θ, z)
    └─ NO  → 직교좌표 (x, y, z)

원점 기준 구 모양인가?
    └─ YES → 구면좌표 (ρ, φ, θ)
```

## 6. 벡터함수의 극한 (Limit of Vector Function)

### 개념

벡터함수 $\vec{r}(t)$는 스칼라 $t$를 입력받아 **벡터를 출력**하는 함수.

$$\vec{r}(t) = \langle f(t),\, g(t),\, h(t) \rangle = f(t)\,\vec{i} + g(t)\,\vec{j} + h(t)\,\vec{k}$$

### 🔑 극한 — 핵심 공식

$$\boxed{\lim_{t \to a} \vec{r}(t) = \left\langle \lim_{t \to a} f(t),\; \lim_{t \to a} g(t),\; \lim_{t \to a} h(t) \right\rangle}$$

> 💡 **핵심 요령**: 벡터함수의 극한은 **각 성분별로 따로따로** 극한을 구하면 된다. 한꺼번에 풀려고 하지 말고 $x$, $y$, $z$ 성분을 분리해서 계산!

**연속 조건:**

$$\vec{r}(t)\text{가 }t=a\text{에서 연속} \iff \lim_{t \to a}\vec{r}(t) = \vec{r}(a)$$

### 미분 (Derivative)

$$\vec{r}\,'(t) = \langle f'(t),\, g'(t),\, h'(t) \rangle$$

- 기하학적 의미: $t$에서의 **접선벡터(Tangent Vector)**
- 단위접선벡터: $\vec{T}(t) = \dfrac{\vec{r}\,'(t)}{|\vec{r}\,'(t)|}$

## 7. 매개변수 방정식 (Parametric Equations)

### 개념

$x$, $y$, $z$ 좌표를 **매개변수 $t$의 함수**로 표현한 것.

$$x = f(t), \quad y = g(t), \quad z = h(t)$$

- $t$는 보통 **시간** 또는 **경로의 위치**를 나타냄
- 매개변수를 소거하면 일반 방정식 $F(x, y) = 0$을 얻을 수 있음

### 🔑 매개변수 소거 요령 — 시험 단골

**📌 삼각함수 항등식 활용** ($\sin^2 t + \cos^2 t = 1$)

**예시**: $x = \cos t$, $y = \sin t$

$$x^2 + y^2 = \cos^2 t + \sin^2 t = 1 \quad \Rightarrow \quad \text{원}$$

**📌 직접 대입법**

**예시**: $x = t$, $y = t^2$

$$t = x \text{를 대입} \quad \Rightarrow \quad y = x^2 \quad \text{(포물선)}$$

### ✏️ 언제 매개변수 방정식을 쓰나?

| 상황 | 이유 |
|------|------|
| 곡선의 **방향·속도** 표현 | $t$가 증가할수록 점이 어느 방향으로 이동하는지 알 수 있음 |
| 일반 방정식으로 표현하기 **어려운 곡선** | 리사주 곡선, 사이클로이드 등 |
| 물체의 **운동 경로** 표현 | 포사체 운동, 궤도 계산 |

### 매개변수 곡선의 기울기

$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{g'(t)}{f'(t)}$$

$$\frac{d^2y}{dx^2} = \frac{d}{dx}\!\left(\frac{dy}{dx}\right) = \frac{\dfrac{d}{dt}\!\left(\dfrac{dy}{dx}\right)}{dx/dt}$$

### ⭐ 호의 길이 (Arc Length) — 자주 출제

$$\boxed{L = \int_a^b \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2 + \left(\frac{dz}{dt}\right)^2}\; dt = \int_a^b |\vec{r}\,'(t)|\, dt}$$

## 8. 직선의 벡터방정식 (Vector Equation of a Line)

### 개념

3차원 공간의 직선은 **한 점 + 방향벡터**로 완전히 결정된다.

$$\boxed{\vec{r}(t) = \vec{r}_0 + t\,\vec{v}}$$

$$\langle x,\, y,\, z \rangle = \langle x_0,\, y_0,\, z_0 \rangle + t\langle a,\, b,\, c \rangle$$

| 기호 | 의미 |
|------|------|
| $\vec{r}_0 = (x_0, y_0, z_0)$ | 직선 위의 한 점 |
| $\vec{v} = \langle a, b, c \rangle$ | 방향벡터 |
| $t$ | 매개변수 (실수 전체) |

### 매개변수 방정식으로 표현

$$x = x_0 + at, \quad y = y_0 + bt, \quad z = z_0 + ct$$

### 🔑 대칭 방정식 (Symmetric Equations) — $t$ 소거의 결과

$t$를 소거하면:

$$\boxed{\frac{x - x_0}{a} = \frac{y - y_0}{b} = \frac{z - z_0}{c}}$$

> ⚠️ 분모가 0인 경우 (예: $a = 0$) → 해당 좌표는 상수로 따로 표기  
> 예: $a = 0$이면 $x = x_0$, 나머지만 대칭식으로

### ✏️ 직선의 방정식 구하는 절차 🔑

```
① 직선 위의 점 (x₀, y₀, z₀)을 찾는다
② 방향벡터 v = ⟨a, b, c⟩를 찾는다
   - 두 점이 주어진 경우: v = P₂ - P₁
   - 두 평면의 교선인 경우: v = n₁ × n₂ (두 법선벡터의 외적)
③ 벡터방정식 r(t) = r₀ + tv에 대입
④ 필요 시 매개변수 방정식 또는 대칭 방정식으로 변환
```

**예시** — 두 점 $P_1(1, 2, 3)$, $P_2(3, 4, 5)$를 지나는 직선:

$$\vec{v} = P_2 - P_1 = \langle 2, 2, 2 \rangle$$

$$\vec{r}(t) = \langle 1, 2, 3 \rangle + t\langle 2, 2, 2 \rangle$$

$$\frac{x-1}{2} = \frac{y-2}{2} = \frac{z-3}{2}$$

## 9. 매개변수화 (Parametrization)

### 개념

곡선이나 곡면을 **매개변수 $t$ (또는 $u$, $v$)를 이용해 표현**하는 것.

### 🔑 주요 곡선의 매개변수화

| 도형 | 매개변수화 | 범위 |
|------|-----------|------|
| 원 (반지름 $r$) | $x = r\cos t,\; y = r\sin t$ | $0 \leq t \leq 2\pi$ |
| 타원 | $x = a\cos t,\; y = b\sin t$ | $0 \leq t \leq 2\pi$ |
| 직선 (두 점 $P_1$, $P_2$) | $\vec{r}(t) = (1-t)P_1 + tP_2$ | $0 \leq t \leq 1$ |
| 포물선 | $x = t,\; y = t^2$ | $t \in \mathbb{R}$ |
| 나선 (Helix) | $x = \cos t,\; y = \sin t,\; z = t$ | $t \in \mathbb{R}$ |

### 🔑 곡면의 매개변수화

곡면은 매개변수가 **두 개** 필요하다.

$$\vec{r}(u, v) = \langle x(u,v),\; y(u,v),\; z(u,v) \rangle$$

**예시 — 구면 (반지름 $a$)**:

$$\vec{r}(\phi, \theta) = \langle a\sin\phi\cos\theta,\; a\sin\phi\sin\theta,\; a\cos\phi \rangle$$

**예시 — 원기둥 (반지름 $a$, 높이 $h$)**:

$$\vec{r}(\theta, z) = \langle a\cos\theta,\; a\sin\theta,\; z \rangle, \quad 0 \leq z \leq h$$

### ✏️ 매개변수화 절차

```
① 도형의 모양을 파악한다 (원, 구, 원기둥, 곡선 등)
② 적절한 매개변수를 선택한다
   - 원형 → 각도 θ 사용
   - 구형 → 구면좌표 φ, θ 사용
   - 직선·곡선 → t 사용
③ 각 좌표를 매개변수로 표현한다
④ 매개변수의 범위를 명시한다
```

> ⭐ **매개변수화가 중요한 이유**
> - 선적분, 면적분 계산 시 **반드시** 매개변수화가 필요하다
> - 복잡한 도형을 단순한 변수로 표현할 수 있어 계산이 쉬워진다
> - 📌 매개변수 **범위 명시**를 빠뜨리면 감점되므로 항상 적을 것!

### 이차곡면 판별표

| 방정식 형태 | 곡면 이름 |
|------------|-----------|
| $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}+\dfrac{z^2}{c^2}=1$ | 타원면 |
| $z=\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}$ | 타원포물면 |
| $z=\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}$ | 쌍곡포물면 (안장) |
| $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=\dfrac{z^2}{c^2}$ | 타원뿔면 |
| $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}-\dfrac{z^2}{c^2}=1$ | 일엽쌍곡면 |
| $-\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}+\dfrac{z^2}{c^2}=1$ | 이엽쌍곡면 |

### 좌표계 변환 요약

| 변환 | 공식 |
|------|------|
| 직교 → 원주 | $r=\sqrt{x^2+y^2}$, $\theta=\arctan(y/x)$, $z=z$ |
| 직교 → 구면 | $\rho=\sqrt{x^2+y^2+z^2}$, $\phi=\arccos(z/\rho)$, $\theta=\arctan(y/x)$ |
| 원주 적분 야코비안 | $dV = r\,dr\,d\theta\,dz$ |
| 구면 적분 야코비안 | $dV = \rho^2\sin\phi\,d\rho\,d\phi\,d\theta$ |

### 직선 방정식 요약

| 형태 | 공식 |
|------|------|
| 벡터방정식 | $\vec{r}(t) = \vec{r}_0 + t\vec{v}$ |
| 매개변수 방정식 | $x=x_0+at,\; y=y_0+bt,\; z=z_0+ct$ |
| 대칭 방정식 | $\dfrac{x-x_0}{a}=\dfrac{y-y_0}{b}=\dfrac{z-z_0}{c}$ |
