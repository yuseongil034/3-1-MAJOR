# 📡 신호 및 시스템 (Signals and Systems)

---

## 1. 신호의 분류 (Classification of Signals)

### 연속시간 vs 이산시간

연속시간 신호는 무한한 정보를 담고 있어 실제로 처리하기 어렵다.
따라서 일정 시간 간격 Ts로 샘플링하여 **이산시간 신호**로 변환하는 것이 핵심이다.
```
x[n] = x(nTs),  n ∈ ℤ
```

---

### 우함수 vs 기함수 (Even vs Odd)

| 종류 | 조건 |
|------|------|
| 우함수 (Even) | x(-t) = x(t) |
| 기함수 (Odd) | x(-t) = -x(t) |

---

### 주기 vs 비주기 (Periodic vs Aperiodic)

**연속시간**
```
x(t) = x(t + T)
```

**이산시간**
```
x[n] = x[n + N],  n, N ∈ ℤ
Ω = 2π / N
```

---

### 결정 vs 무작위 (Deterministic vs Random)

| 종류 | 설명 |
|------|------|
| 결정 신호 | 수식으로 정확히 표현 가능 |
| 무작위 신호 | 확률적으로만 기술 가능 |

---

### ⚡ 에너지 vs 파워 신호 (Energy vs Power)

**에너지 신호**
```
E = ∫ |x(t)|² dt   (적분 범위: -∞ ~ +∞)
```

**파워 신호**
```
P = (1/T) ∫ |x(t)|² dt   (한 주기 T 구간 적분)
```

> 💡 E가 발산해도 P가 유한하면 파워 신호이다.
> 에너지 신호와 파워 신호는 동시에 될 수 없다.
> 주기 신호 → 대부분 파워 신호 / 비주기 신호 → 대부분 에너지 신호

---

## 2. 신호의 연산 (Operations on Signals)

### 종속변수 연산 (Dependent Variable Operations)

| 연산 | 표현 |
|------|------|
| 진폭 스케일링 (Amplitude Scaling) | y(t) = A · x(t) |
| 덧셈 (Addition) | y(t) = x₁(t) + x₂(t) |
| 곱셈 (Multiplication) | y(t) = x₁(t) · x₂(t) |
| 미분 | y(t) = dx(t)/dt |
| 적분 | y(t) = ∫ x(τ) dτ |

---

### 독립변수 연산 (Independent Variable Operations)

| 연산 | 표현 | 설명 |
|------|------|------|
| 시간 스케일링 (Time Scaling) | y(t) = x(at) | a>1 압축 / 0<a<1 확장 |
| 반사 (Reflection) | y(t) = x(-t) | 좌우 대칭 |
| 시간 이동 (Time Shifting) | y(t) = x(t - t₀) | t₀>0 오른쪽 / t₀<0 왼쪽 |

---

## 3. 기본 신호 (Elementary Signals)

### 지수 신호 (Exponential Signals)
```
연속시간:  x(t) = B·eᵃᵗ
이산시간:  x[n] = B·rⁿ
```

---

### 정현파 신호 (Sinusoidal Signals)

**연속시간** → 무조건 주기 신호
```
x(t) = A·cos(ωt + φ)
```

**이산시간** → 주기 신호가 될 수도, 아닐 수도 있음
```
x[n] = A·cos(Ωn + φ)
```

> 💡 이산시간 정현파가 주기 신호가 되려면 아래 조건을 만족해야 한다.
> ```
> ΩN = 2πk   (N, k ∈ ℤ)
> ```
> 즉, Ω/2π 가 **유리수**이면 주기 신호이다.

---

### 지수 감쇠 신호 (Exponentially Damped Sinusoidal)
```
x(t) = A·e^(-αt) · sin(ωt + φ)
```

---

### 단위 계단 함수 (Unit Step Function)

**이산시간**
```
u[n] = 1  (n ≥ 0)
u[n] = 0  (n < 0)
```

**연속시간**
```
u(t) = 1  (t > 0)
u(t) = 0  (t < 0)
```

---

### 임펄스 함수 (Impulse Function)

**이산시간**
```
δ[n] = 1  (n = 0)
δ[n] = 0  (n ≠ 0)
```

**연속시간**
```
δ(t) = 0  (t ≠ 0)
∫ δ(t) dt = 1   (적분 범위: -∞ ~ +∞)
```

**주요 성질**

| 성질 | 수식 |
|------|------|
| 우함수 | δ(-t) = δ(t) |
| 시프팅 (Sifting) | ∫ x(t)·δ(t - t₀) dt = x(t₀) |
| 시간 스케일링 | δ(at) = δ(t) / \|a\| |

---

### 램프 함수 (Ramp Function)

**연속시간**
```
r(t) = t   (t ≥ 0)
r(t) = 0   (t < 0)
```

**이산시간**
```
r[n] = n   (n ≥ 0)
r[n] = 0   (n < 0)
```

---

### 🔗 세 함수의 관계
```
적분 방향 →
δ(t)  →  u(t)  →  r(t)

미분 방향 ←
r(t)  →  u(t)  →  δ(t)
```

> 임펄스를 적분하면 계단, 계단을 적분하면 램프
> 램프를 미분하면 계단, 계단을 미분하면 임펄스
